## Event Handler in react:
A function that runs when a user performs an action (click, type, submit, etc.).
* In html we use, <button onclick="handleClick()">Click</button>
* But in react we use, <button onClick={handleClick}>Click</button>

##in react we use suspense and use to load the data from api.
##suspense:
<Suspense> Suspense is a React component that shows a loading UI while something is loading.

Ex:
<Suspense fallback={<Loading..... />}>
  <Users />
</Suspense>
fallback → What users see while waiting.
<Users /> → The component that is loading.
